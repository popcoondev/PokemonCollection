# FireRed攻略モード 仕様メモ

## 目的
- ポケモン図鑑とは別に、`ポケットモンスター ファイアレッド` 攻略に特化した情報閲覧モードを提供する
- 既存図鑑は汎用図鑑のまま維持し、FireRed 固有情報は専用モードへ分離する

## メニュー
- メニュー画面に `FR攻略` を追加する
- `FR攻略` へ入ると、以下の入口を表示する
  - `ポケモンからみる`
  - `場所からみる`

## ポケモンからみる
- 全国図鑑順のポケモン名リストを表示する
- 1ページあたり 10 件表示
- 左右ページ送りで次/前ページへ移動できる
- リスト項目を開くと、FireRed 専用のポケモン詳細画面へ遷移する

### FRポケモン詳細
- 見た目は既存図鑑に近い
- タブは攻略寄りにする
  - `しゅつげん`
  - `しんか`
  - `わざ`
  - `マシン`
  - `ひでん`
- 右上に `つかまえた` の ON/OFF トグルを置く
- トグルはページ送り操作と重ならない位置に置く
- 捕獲済み状態は FireRed 攻略専用の設定として保存する

### タブ内容
- `しゅつげん`
  - 場所名
  - 出現方法
  - 出現率
- `しんか`
  - 条件文字列をそのまま表示
  - 例: `Lv16`, `ほのおのいし`, `つうしん`
- `わざ`
  - 最初はレベル技のみ
  - `Lv` と技名の一覧
- `マシン`
  - 覚えられるわざマシン一覧
- `ひでん`
  - 覚えられるひでんわざ一覧

## 場所からみる
- ゲーム進行順の場所リストを表示する
- 1ページあたり 10 件表示
- 左右ページ送りで次/前ページへ移動できる
- 場所を開くと、その場所で出現するポケモン一覧を表示する
- 一覧のポケモンを開くと、FRポケモン詳細へ遷移する

## データ配置
- FireRed 固有情報は既存図鑑 JSON と分離する

### ポケモン別データ
- `/pokemon/firered/pokemon/%04d.json`

想定例:

```json
{
  "id": 1,
  "locations": [
    { "area": "トキワのもり", "method": "くさむら", "rate": "5%" }
  ],
  "evolution": [
    "Lv16"
  ],
  "level_up_moves": [
    { "level": 1, "name": "たいあたり" },
    { "level": 7, "name": "やどりぎのタネ" }
  ],
  "machines": ["01 きあいパンチ"],
  "hms": ["01 いあいぎり"]
}
```

### 場所一覧データ
- `/pokemon/firered/locations.json`

想定例:

```json
[
  {
    "id": "route_01",
    "name": "1ばんどうろ",
    "pokemon": [16, 19]
  }
]
```

### 捕獲済み状態
- `/pokemon/firered/caught.json`
- `id -> true/false` の単純な保存でよい

## 実装順
1. `FR攻略` メニュー追加
2. `ポケモンからみる` 一覧
3. FRポケモン詳細の骨組み
4. 捕獲済みトグル保存
5. `しゅつげん` / `しんか` / `わざ`
6. `マシン` / `ひでん`
7. `場所からみる` 一覧
8. 場所詳細
