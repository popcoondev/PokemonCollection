# 課題メモ

動作確認で見つかった課題をこのファイルに追記して対処する。

## 使い方

- 1課題につき1項目で記録する
- `状態` は `open` / `doing` / `done` を使う
- 再現手順と期待結果を短く残す
- 修正後は `対応メモ` に変更内容を書く

## テンプレート

### 0001. 課題タイトル
- 状態: open
- 発生日: YYYY-MM-DD
- 画面/機能:
- 再現手順:
- 期待結果:
- 実際の結果:
- 対応メモ:
